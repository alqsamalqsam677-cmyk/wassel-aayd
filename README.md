# Wassel Aayd

منصة وساطة وتحويل آمنة لليمنيين.

## فكرة المشروع
ربط اليمنيين في الخارج مع أهلهم في الداخل عبر وسطاء موثوقين.

## المميزات الأولية
- تسجيل مستخدم
- إنشاء طلب
- قبول الطلب من وسيط
- تتبع حالة الطلب
- نظام تقييم

## الهدف
تقليل الاحتيال وبناء الثقة في السوق اليمني.مني.
